## Questions
1. Which simple firewall tool is common on Ubuntu?
   - A) firewalld
   - B) ufw
   - C) iptables
   - D) pf
   <details>
   <summary>Answer</summary>
   ufw
   </details>

2. Where do you change the SSH port?
   - A) /etc/ssh/sshd_config
   - B) /etc/passwd
   - C) /etc/hosts
   - D) /etc/fstab
   <details>
   <summary>Answer</summary>
   /etc/ssh/sshd_config
   </details>

3. Where are Netplan configs stored?
   - A) /etc/netplan/
   - B) /etc/network/
   - C) /etc/sysconfig/
   - D) /etc/ifcfg/
   <details>
   <summary>Answer</summary>
   /etc/netplan/
   </details>

4. Preferred secure access method for production?
   - A) Root password
   - B) Key-based SSH
   - C) Open ports
   - D) Telnet
   <details>
   <summary>Answer</summary>
   Key-based SSH
   </details>
