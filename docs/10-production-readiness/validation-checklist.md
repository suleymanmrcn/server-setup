Uretime cikmadan once minimum dogrulama listesi.

## Checklist
- [ ] SSH anahtar tabanli erisim
- [ ] Root login kapali
- [ ] Firewall default deny
- [ ] Monitoring aktif
- [ ] Backup ve restore testi yapildi
- [ ] Patch penceresi tanimli
- [ ] Log rotasyonu aktif

## Not
Eksik madde varsa uretime cikma.
