## Sorular
1. Rocky/Alma Linux hangi aileye yakindir?
   - A) Debian
   - B) RHEL
   - C) Arch
   - D) SUSE
   <details>
   <summary>Cevap</summary>
   RHEL
   </details>

2. Varsayilan firewall servisi hangisidir?
   - A) ufw
   - B) firewalld
   - C) pf
   - D) iptables
   <details>
   <summary>Cevap</summary>
   firewalld
   </details>

3. Guvenli SSH icin hangisi onerilir?
   - A) Root login
   - B) Anahtar tabanli erisim
   - C) Sifreyi paylas
   - D) Telnet
   <details>
   <summary>Cevap</summary>

   </details>
