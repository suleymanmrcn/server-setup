## Sorular
1. RHEL icin resmi repository erisimi neyle yonetilir?
   - A) Subscription Manager
   - B) APT
   - C) Snap
   - D) Flatpak
   <details>
   <summary>Cevap</summary>
   Subscription Manager
   </details>

2. RHEL'de kernel guncellemeleri icin tercih edilen yontem?
   - A) Rastgele reboot
   - B) Planli bakim penceresi
   - C) Guncelleme yok
   - D) Sadece GUI
   <details>
   <summary>Cevap</summary>
   Planli bakim penceresi
   </details>

3. RHEL hardening icin hangi politika onemlidir?

   - B) AppArmor profile
