---
title: Oracle Baglanti Modelleri (Peering, Private Endpoint)
sidebar_position: 14
---

Oracle Cloud tarafinda VCN baglanti secimlerini ozetler.

## VCN Peering
- Local veya Remote Peering
- CIDR cakismasi olmamali

## Service Gateway
- Oracle servislerine ozel erisim
- Internet olmadan servis kullanimi

## Private Endpoint
- Servis bazli ozel erisim
- Public IP kullanmadan baglanti

## Ne Zaman Hangisi?
- Iki VCN: Peering
- Oracle servisleri: Service Gateway
- Servis erisimi: Private Endpoint
